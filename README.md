# Canoryn Issues & Feedback

This repository is the public issue tracker for **Canoryn**.

Use it to:

1. Report bugs
2. Request features
3. Share product feedback

The application source code is private and is not hosted here.

## Before filing an issue

1. Search existing issues first.
2. Include app version and macOS version.
3. Share clear steps to reproduce.
4. Include expected vs actual behavior.
5. Attach logs/screenshots when possible.

## Quick links

- Report a bug: [New bug report](https://github.com/procodeai/canoryn-issues/issues/new?template=bug_report.yml)
- Request a feature: [New feature request](https://github.com/procodeai/canoryn-issues/issues/new?template=feature_request.yml)
- View all issues: [Issues](https://github.com/procodeai/canoryn-issues/issues)
